# PAYE Tax Calculation System

A Java-based Pay As You Earn (PAYE) tax calculation system developed for PERLONGS Computing Services.

## Features
- Calculate PAYE tax based on progressive tax brackets
- Tax-free threshold: First K4,000
- Multiple tax brackets up to 37%
- NAPSA contribution calculation (5% with cap)
- Professional payslip generation
